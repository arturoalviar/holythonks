# HolyThonks

A collection of current images from [r/MechanicalKeyboards](https://www.reddit.com/r/MechanicalKeyboards/) and [r/CustomKeyboards](https://www.reddit.com/r/CustomKeyboards/)

Built with Next.js and Chakra UI

## Getting Started

```bash

yarn run dev

```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
